# GLOSSARY.md — 공유 용어집

| 용어 | 정의 |
|---|---|
| 천장 (Ceiling) | 가중치·학습으로 정해진 모델 능력의 한계. 프롬프트로 못 올린다 |
| 발현 (Elicitation) | 천장 중 실제로 끌려 나오는 능력의 양. 추론 구조·깊이로 달라진다. 이 팩의 대상 |
| test-time compute | 추론 시 쓰는 연산량(추론 토큰의 양). 늘리면 어려운 문제 정확도가 오른다. 내부에서만 작동 |
| 그라운딩 (Grounding) | 주장을 외부 근거(출처·도구·계산)에 붙이는 것. 사실성의 실질 담보 |
| 자기검증 (Self-critique) | 모델이 자기 답을 내부에서 비판·수정. 자신이 탐지 가능한 오류만 잡힘(한계 있음) |
| 발신 전 점검 | 보내기 전 내부 4점검(사실성·논리·출처·커버리지). 출력하지 않음 |
| 불변식 (Invariant) | 1건 위반 시 발신 금지 규칙(`INVARIANTS.md`) |
| 네이티브 스타일 | 기본 Opus 4.8/Fable 5의 말투·문체·형식. 이 하네스가 바꾸지 않는 것 |
