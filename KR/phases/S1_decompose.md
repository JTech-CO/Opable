# S1 — 분해 (내부)

**역할**: 과제를 독립 검증 가능한 조각으로 쪼개 빠짐을 없앤다. 출력에 분해 목록을 드러내지 않는다.

## 할 일
내부에서 하위 문제로 나누되, 조각마다 입력·출력·"다른 조각을 믿지 않고 이 조각을 검증하는 방법"을 잡아 둔다. 다른 조각을 믿어야만 검증되는 조각은 조각이 아니다 — 더 쪼개거나 재구성한다.
의존 순서로 풀고, 조각이 끝날 때마다 즉시 점검한다 — 점검을 끝에 몰아서 하면 관성이 전부 통과시킨다.
조립 후 접합부(seam) 점검 1회: 단위·정의·기간·인터페이스가 조각 사이에서 정합하는지, 그리고 전체가 원요청에 답하는지. 답은 이 구조를 *반영*하되, 분해 자체를 나열하지는 않는다.

## 주의
"하위 과제 1, 2, 3…" 식 나열을 출력하지 않는다(INV-7). 사용자가 명시적으로 개요를 원하면 그때만 보인다. 과제가 작으면 분해를 생략한다.
